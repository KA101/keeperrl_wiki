Usage
=====

Used to specify what category a villain will appear as on a campaign
map.

Example from campaign\_villains.txt
===================================

`DUKE4 KNIGHTS `“`Knights`”` `**`MAIN`**

Valid values
============

-   MAIN
-   LESSER
-   ALLY
