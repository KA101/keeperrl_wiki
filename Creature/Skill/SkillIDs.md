Usage
=====

Used to specify what special skills immigrants may have.

Example from immigration.txt
============================

`\{ 0.03 \{ SkillId `**`WORKSHOP`**` LastingEffect INSANITY \}\}`

Valid values
============

-   AMBUSH
-   STEALING
-   SWIMMING
-   DIGGING
-   NAVIGATION\_DIGGING
-   DISARM\_TRAPS
-   SORCERY
-   CONSUMPTION
-   COPULATION
-   CROPS
-   SPIDER
-   STEALTH
-   WORKSHOP
-   FORGE
-   LABORATORY
-   JEWELER
-   FURNACE
-   EXPLORE
-   EXPLORE\_NOCTURNAL
-   EXPLORE\_CAVES
