Usage
=====

Used to specify what types of attributes special immigrants may have

Example from immigration.txt
============================

`specialTraits = \{`  
`  \{ 0.03 \{ AttrBonus `**`SPELL_DAMAGE`**` 4 AttrBonus `**`DEFENSE`**` -4\}\}`  
`\}`

Valid values
============

-   DAMAGE
-   SPELL\_DAMAGE
-   RANGED\_DAMAGE
-   DEFENCE
