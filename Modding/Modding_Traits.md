Usage
=====

Used to specify what traits immigrants have

Example from immigration.txt
============================

`traits = \{ `**`FIGHTER`` ``NO_AUTO_EQUIPMENT`**` \}`

Valid Values
============

-   LEADER
-   FIGHTER
-   WORKER
-   PRISONER
-   NO\_EQUIPMENT
-   NO\_LIMIT
-   FARM\_ANIMAL
-   SUMMONED
-   NO\_AUTO\_EQUIPMENT
-   DOESNT\_TRIGGER
-   INCREASE\_POPULATION
