Usage
=====

Used to specify various immigration options.

Example from immigration.txt
============================

**`noAuto`**` = true`  
**`initialRecruitment`**` = 4`

Valid values
============

-   invisible
-   noAuto
-   consumeIds
-   initialRecruitment
-   frequency
